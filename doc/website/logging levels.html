Logging levels

voir les niveaux python. http://wiki.ops4j.org/confluence/display/ops4j/Pax+Logging:

In the case of JDK Logging there is a mapping occuring between the Levels in JDK Logging and in Log4J. The definition chosen is as follows;

JDK Logging Level Log4J Level Comment
OFF OFF
WARNING WARN
SEVERE ERROR
INFO INFO
FINE DEBUG
FINER TRACE
FINEST TRACE
CONFIG n/a CONFIG events are ignored.
n/a FATAL No such thing in JDK Logging.
