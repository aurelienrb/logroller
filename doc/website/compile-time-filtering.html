Compile-time filtering

Disabling a call to a C style function at compile time can be difficult if it is a variadic one: 

#define TRACE printf

void Test()
{
    TRACE( "Entering %s", __FUNCTION__ );
}

On Windows with Microsoft Visual C++, the __noop compiler intrinsic can be used to ignore the TRACE call and its arguments: 

#define TRACE __noop
