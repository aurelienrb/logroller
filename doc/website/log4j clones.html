Log4j clones

JS: http://log4js.sourceforge.net/ LUA: http://www.keplerproject.org/lualogging/ PHP: http://logging.apache.org/log4php/ log4net: http://logging.apache.org/log4net/

Python standard logging library was inspired of it

http://www.nlog-project.org/: The API (application programming interface) is similar to log4net.

NLog supports .NET, C/C++ and COM interop API so that all your application components including legacy modules written in C++/COM can send their messages through a common log routing engine.
